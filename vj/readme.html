Installation: Den berömda 5-minutersinstallationen

  1. Packa upp zip-filen i en tom katalog och ladda upp hela filstrukturen till din webb-server.
  2. Öppna sidan wp-admin/install.php i webbläsaren. Detta kommer att ta dig genom processen för att skapa filen wp-config.php som innehåller den nödvändiga informationen för anslutning till webbplatsens underliggande databas.
    1. Var inte orolig om den automatiska installationen av någon anledning inte fungerar. Sånt kan hända på vissa webbhotell. Öppna filen wp-config-sample.php i ett textredigeringsprogram, t.ex. WordPad eller liknande, och fyll i informationen om din databasanslutning.
    2. Spara sedan filen under namnet wp-config.php och ladda upp den till webbhotellet.
    3. Öppna wp-admin/install.php i webbläsaren.
  3. När konfigurationsfilen är färdig kommer installationsproceduren att bygga de databastabeller som din blogg behöver. Om något blir fel bör du kontrollera filen wp-config.php en extra gång och försöka igen. Om det fortfarande inte skulle fungera är du välkommen att besöka support-forumet och berätta om ditt problem. Förbered all information som kan vara till hjälp.
  4. Om du inte angav något lösenord kommer systemet att ge dig ett. Se till att spara det. Om du inte angav något användarnamn, kommer det att vara admin.
  5. När allt är klart bör installationsprogrammet skicka dig vidare till inloggningssidan. Logga in med ditt användarnamn och lösenord som du skapade under installationen. Om systemet skapade ett lösenord åt dig kan du klicka på Användare – Din profil för att ändra lösenordet.

Uppdateringar

Användning av den automatiska uppdateringen

Om du uppdaterar från version 2.7 eller senare kan du använda den automatiska uppdateringen:

  1. Öppna wp-admin/update-core.php i din webbläsare och följ instruktionerna.
  2. Svårare än så är det inte!

Manuell uppdatering

  1. Se först till att du har säkerhetskopior på alla filer som du eventuellt kan ha ändrat, t.ex. index.php.
  2. Radera dina gamla WordPress-filer, men spara de du gjort ändringar i.
  3. Ladda upp de nya filerna.
  4. Öppna webbläsaren och gå till /wp-admin/upgrade.php.

Att flytta till WordPress från andra system

WordPress kan importera innehåll från en rad olika system. Börja med att installera och konfigurera WordPress enligt beskrivningen ovan. Sedan kan du använda våra importverktyg.

Systemkrav

  • PHP version 5.6.20 eller senare.
  • MySQL version 5.0 eller senare.

Rekommenderas

  • PHP version 7.3 eller senare.
  • MySQL version 5.6 eller senare.
  • Apache-modulen mod_rewrite.
  • Stöd för HTTPS.
  • Länka gärna till wordpress.org från din webbplats.

Online-resurser

Om du undrar över något som inte besvaras i det här dokumentet kan du söka bland all online-information som finns för WordPress:

WordPress Codex
Codex är encyklopedin för allt som rör WordPress. Det är den mest omfattande informationskällan som finns för WordPress.
WordPress svenska blogg
Här publiceras bl.a. information om nya versioner av WordPress. Om du använder WordPress på svenska kan du hitta länkar hit från admin-panelen för din WordPress-webbplats.
WPSV
WPSV är hemmabasen för den svenska communityn kring WordPress. Vi underhåller den svenska versionen av WordPress, ordnar WordPress-träffar, hjälper varandra med support o.s.v. Allt arbete sker på frivillig väg, och du är välkommen att också hjälpa till.
Den officiella WordPress-bloggen
Här hittar du den senaste officiella informationen om WordPress och dess uppdateringar (på engelska). Om du använder WordPress på engelska kan du hitta denna information i din egen admin-panel.
WordPress Planet
WordPress Planet är en webbplats som samlar artiklar och inlägg om WordPress från en rad olika bloggar runtom i världen (på engelska).
WordPress Supportforum
Om du letat överallt men ändå inte hittat svar på din fråga kan du ställa din fråga till andra WordPress-användare i forumet. Du har störst chans att få hjälp om du ser till att sätta en tydlig, beskrivande rubrik på din fråga och inkludera så detaljerad information som möjligt och om vad du redan har provat. Länken ovan går till det svenska forumet. Det internationella forumet hittar du på https://wordpress.org/support/.
WordPress IRC-kanal
Det finns även en chatt-kanal för diskussioner kring WordPress, där man ibland tar upp supportärenden. Länken här ovan leder till en sida med mer information om detta. (irc.freenode.net #wordpress)

Några ord på vägen

  • Om du har några förslag, idéer eller kommentarer eller (Ajdå!) har hittat något fel är du välkommen till support-forumet.
  • WordPress har ett rejält API för tillägg, vilket gör det enkelt att utöka koden med nya funktioner. Om du är utvecklare och funderar på att använda detta rekommenderar vi att du läser Plugin Developer Handbook. Undvik att göra ändringar direkt i filerna i WordPress-kärnan.
